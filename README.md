# Customer_behaviour_analysis

Data Analytics Project – End-to-End Pipeline
Overview

This project demonstrates a complete data analytics workflow — from importing raw data into Python to delivering final business insights through SQL, Power BI, and automated reporting.
It highlights core analytics skills including EDA, data cleaning, database querying, dashboard creation, and presentation/report automation.


Project Steps
1. Data Loading (Python)
Loaded raw data using Pandas.
Inspected structure, columns, datatypes
Verified missing values and basic statistics

2. Exploratory Data Analysis (EDA)
Distribution analysis
Trend and correlation analysis
Outlier detection
Key visualizations for understanding patterns

3. Data Cleaning & Preparation
Handling missing values
Removing duplicates
Fixing data types
Exporting cleaned dataset for SQL usage

4. SQL Queries (PostgreSQL / SQL Server)
Imported cleaned dataset into DB
Performed: Filtering & sorting,Joins, Aggregations,Window functions,Business logic queries
Extracted results for visualization and reporting

5. Power BI Dashboard
Connected dataset/SQL results
Modeled relationships
Built interactive visuals:
KPIs
Bar/line charts
Filters & slicers
Designed a clean, business-ready dashboard

6. Report & Presentation (Gamma)
Created an automated slide deck summarizing:
Problem statement
Data approach
Findings
Visual insights

7. Recommendations
Generated a polished PDF/PPT for stakeholders
Dashboard & Results
The final Power BI dashboard highlights:
Key performance indicators (KPIs)
Trends and relationships
Actionable business insights
